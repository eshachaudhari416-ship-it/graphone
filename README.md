# GraphOne Frontend

Pixel-perfect UI clone of GraphOne — the global intelligence layer for the AI economy.

## 🚀 Live Site
URL: `https://your-vercel-url.vercel.app`

## 📦 Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts (charts)
- Framer Motion (animations)
- Lucide React (icons)
- Deployed on Vercel

## 🖥️ Pages Built
1. **AI Companies Page** — trending, fastest growing, emerging startups, unicorns, browse by category
2. **Company Detail Page** — timeline, funding, ownership chart, investors, founders, products, news, competitors
3. **Investors Page** — trending investors, collections, browse by type, most active
4. **Investor Profile Page** — thesis, portfolio concentration, recent investments
5. **AI Products Page** — category tabs, popular products, collection of the week

## 🔧 Local Setup

### 1. Clone the repo
```bash
git clone https://github.com/eshachaudhari416-ship-it/graphone.git
cd graphone
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

Open `http://localhost:3000` in your browser.
